# Portfolio 2

## Goal

Plan and carry out a professional presentation of a portfolio for potential clients or employers.

## Description

Expand on your portfolio 1 assignment by developing a multipage website. Include the following projects in your portfolio website:

- CSS Frameworks
- JavaScript Frameworks
- Semester Project 2

You must make at least one improvement to each of the projects listed above. Review and refine each project and prepare it to be shown professionally in your portfolio. Each project should appear as a teaser or summary on the main portfolio page. When a user selects a project, it should open a dedicated article page with full details.

### Feature requirements

- Home page with three teaser cards
- Three article pages, one for each teaser

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- React Router
- React Helmet

## Installing

1. Clone the repo:

```bash
git clone https://github.com/Jereriviel/project-exam-2.git
```

2. Install dependencies

```bash
npm install
```

3. Start the local dev server:

```bash
npm run dev
```

4. Build the project for production:

```bash
npm run build
```

## Linting and Formatting

Run Prettier:

```bash
npm run format
```

Run ESLint:

```bash
npm run lint
```

## Pre-Commit Hooks with Husky

Husky is configured to automatically check linting and formatting before each commit:

```bash
npm run prepare
```
